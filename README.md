# Real-Time-Messaging-Platform
A real-time messaging platform using Flask and WebSockets, supporting authenticated  users, private and group channels, and low-latency message delivery. Implemented backend services and REST APIs with Flask, handling message persistence, user presence, and  role-based access control, with data stored in MongoDB . 

#How to run it:
1: Install dependencies
pip install -r requirements.txt

2: Start MongoDB
mongod

3:Run the server
python app.py

4:open in Browser
http://localhost:5000
